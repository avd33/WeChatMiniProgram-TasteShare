# 基于微信小程序的美食分享应用
**开发专业**：数字媒体技术
**开发技术**：微信小程序原生框架（WXML + WXSS + JavaScript）
**开发工具**：微信开发者工具
**项目类型**：移动应用开发课程期末大作业
**核心功能**：美食发现、多媒体展示、美食记录、收藏管理、数据本地持久化

## 项目简介
本项目为课程期末大作业，基于微信小程序原生框架开发了一款轻量化美食分享应用。项目整合「美食发现、内容展示、数据管理、个性化收藏」四大核心能力，覆盖用户从浏览美食、查看详情到记录体验、管理收藏的全流程，全程采用组件化开发思想，基于微信本地存储 API 实现数据持久化，兼顾操作流畅性与用户体验。

## 核心功能模块
1. **首页发现模块**：美食分类展示、推荐内容列表、轮播图Banner、下拉刷新/上拉加载更多
2. **美食详情模块**：美食图片预览、图文详情展示、收藏/取消收藏、分享功能
3. **美食记录模块**：结构化表单填写（美食名称、分类、体验描述等）、表单验证、本地保存
4. **个人中心模块**：用户信息展示、收藏列表管理、记录历史查看、功能入口统一管理
5. **通用组件模块**：封装导航栏（navigation-bar）等自定义组件，提升代码复用性

## 关键技术亮点
- 采用微信小程序原生 MINA 框架，遵循「数据驱动视图」设计思想，页面间通信高效
- 基于 `wx.setStorageSync`/`wx.getStorageSync` 实现本地数据持久化，保障数据不丢失
- 自定义组件开发：封装导航栏等通用组件，降低代码冗余，提升维护性
- 列表优化：实现下拉刷新、上拉加载更多，优化大数据列表渲染性能
- 表单验证：覆盖必填项、格式校验等场景，提升用户输入体验

## 运行方式
1. 下载本仓库所有文件至本地；
2. 打开「微信开发者工具」，点击「导入项目」；
3. 选择本地仓库文件夹，填写小程序 AppID（测试可选择「测试号」）；
4. 点击「导入」，即可在模拟器/真机调试模式下运行项目。

## 开发与测试说明
- 开发环境：Windows 11 + 微信开发者工具（最新版）
- 测试结果：所有核心功能运行稳定，界面适配不同手机尺寸，操作流畅
- 待优化方向：可扩展微信云开发实现数据同步、LBS 附近美食推荐、用户评论互动等功能

## 数据环境说明
本小程序运行所需的美食店铺数据依赖提供的专用服务器环境，前端代码已预留数据接口对接逻辑，在指定服务器环境下可正常获取并展示首页店铺数据。
